# Miyagi Menroku

Miyagi Menroku is a web application for logging and managing ramen experiences. It allows users to record ramen shops they’ve visited, track details about each bowl, and build a personal history of ramen discoveries.

---

## Features

* Log ramen shop visits with notes and ratings
* Track key details such as broth type, noodles, and toppings
* Organize and review past ramen experiences
* Search and browse your ramen history
